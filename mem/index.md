# Project Memory

## Core
eDeen = "Empresa para el Desarrollo de Espacios Naturales". Centro de jardinería en Palma desde 2012. C/ Joan Mascaró i Fornés 79.
Tono de marca: anfitrión/invitado — hospitalidad, generosidad, cuidado, respeto, cercanía, honestidad, "buen rollo". Arquetipo: cuidador. Brand equity: desconexión positiva.
Servicios: plantas/macetas/sustratos, paisajismo (diseño+ejecución+mantenimiento), floristería para eventos, cafetería Es Berenar (9–14h diario).
Colores corporativos: #0169af, #33a2ed, #729db9, #b88963 (acento cálido bronce).
Valores: Comprometidos, Auténticos, Buen Rollo, Cercanos. Frase clave: "Te tratamos como a un invitado".
